🇬🇧 English with RSUNG Bot

Description:
This bot helps users learn English interactively by translating words. The user receives a random English word, attempts to translate it, and immediately sees the result.

🚀 Features

Interactive word translation quizzes

Hint button “Show Answer” for quick checking

Personal statistics: attempts, correct and incorrect answers

Supports dictionary file (ENRUS1.TXT) with cp1251 encoding

Integration with RSUNG web application via button

🛠 Technologies

Python 3.10+

pyTelegramBotAPI (telebot) — Telegram Bot API library

random — for random word selection

InlineKeyboardMarkup — interactive buttons in Telegram

TXT dictionary — stores words and translations
